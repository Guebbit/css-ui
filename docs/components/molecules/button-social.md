# Button Social
<Badge type="tip">Molecule</Badge> <Badge type="info">buttons</Badge>
Simple buttons with the right colors included. *theme.create-colors(("brand"))* for brands.

::: tip INCLUDED
- "create-colors" from @guebbit/css-toolkit
- [Button](/components/atoms/button)
- [HighlightPopup](/components/atoms/highlight-popup)
- [HighlightRollup](/components/atoms/highlight-rollup)
:::

## Use
```scss
@use "@guebbit/css-ui/styles" as theme;
@use "@guebbit/css-ui/atoms/button";
@use "@guebbit/css-ui/atoms/highlight-popup";
@use "@guebbit/css-ui/atoms/highlight-rollup";

@include theme.create-colors(theme.$colors-collection, ("border", "pseudo", "hover"));
```

## Other buttons
Check for other buttons and styles. With these classes or the SCSS variables combinations are infinite

## Classes

Uses the documented classes from [Button](/components/atoms/button), [HighlightPopup](/components/atoms/highlight-popup), and [HighlightRollup](/components/atoms/highlight-rollup), plus the brand utility classes generated by `theme.create-colors(...)`.

## SCSS variables

No social-button-specific SCSS variables. Use the variables documented in [Button](/components/atoms/button), [HighlightPopup](/components/atoms/highlight-popup), and [HighlightRollup](/components/atoms/highlight-rollup).
