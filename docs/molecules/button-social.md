# Social buttons
<Badge type="tip">Molecule</Badge> <Badge type="info">buttons</Badge>
Simple buttons with the right colors included. *theme.create-colors(("brand"))* for brands.

::: tip INCLUDED
- "create-colors" from @guebbit/css-toolkit
- [SimpleButton](/atoms/button-simple)
- [ButtonLargeIcon](/atoms/button-large-icon)
:::

## Use
```scss
@use "@guebbit/css-ui/styles" as theme;
@use "@guebbit/css-ui/atoms/button-simple";
@use "@guebbit/css-ui/atoms/button-large-icon";

@include theme.create-colors(theme.$colors-collection, ("border", "pseudo", "hover"));
```

## Other buttons
Check for other buttons and styles. With these classes or the SCSS variables combinations are infinite

## Classes

Uses the documented classes from [SimpleButton](/atoms/button-simple) and [ButtonLargeIcon](/atoms/button-large-icon), plus the brand utility classes generated by `theme.create-colors(...)`.

## SCSS variables

No social-button-specific SCSS variables. Use the variables documented in [SimpleButton](/atoms/button-simple) and [ButtonLargeIcon](/atoms/button-large-icon).
